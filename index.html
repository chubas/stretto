Stretto is a Ruby API for programming music. Based on JFugue, it makes programming music not only easy but fun!

Getting started

Stretto is a Ruby native implementation over the JFugue syntax. It also uses native libraries to handle the MIDI input and output.

Features

  • Write all kind of music elements

    Using JFugue language notation, Stretto allows you to write music notes, chords, instruments, voices and tracks, tempo variations, keys, and other elements.

  • Extensible MIDI interface

    Stretto has a neat separation between the musical notation and the MIDI I/O. This allows to extend or write your own interfaces for input and output. It comes bundled with the Midiator interface, that allows to use external MIDI devices out of the box.

  • JFugue notation syntax

    By using a standarized JFugue syntax, you can use your musical compositions in cross-language projects with any of the Java family languages.

Documentation

To learn more from Stretto classes and the supported JFugue classes, you can check the RDoc documentation. Stretto itself is an adaptation from the JFugue project, you can check the full manual here. This contains more information about the language syntax, and should give you a good approximation to using Stretto classes - they are in fact almost equivalent to their Java counterparts.

You can also browse the source code directly from Github, or download the project and run the included examples.
